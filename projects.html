VOS (Virtual Operating System Framework)

STATUS: concept

A simulated, virtual unix-like operating system written in .NET 8. Useful for ARGs or other kinds of real-life games. It's based off of a real linux filestructure and offers basic read-only access to a pre-setup 'computer', simulating the feeling of connecting to a machine via SSH.
